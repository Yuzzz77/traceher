# 提示词模板

## 让模型改页面

```text
你是我的前端学习助手。我是大二信管专业学生，正在做实习作品集。

请先阅读项目规则文件，然后帮我优化这个页面：

目标：
1. 页面要更适合放进作品集。
2. 保留原有功能。
3. 增加一点粒子或交互亮点。
4. 每一步解释清楚，适合新手理解。

完成后请告诉我：
1. 改了哪些文件。
2. 怎么打开查看。
3. 这个项目可以怎么写进简历。
```

## 让模型做粒子系统

```text
请帮我做一个可直接打开的粒子交互网页。

要求：
1. 使用 HTML、CSS、JavaScript。
2. 粒子要能跟随鼠标产生变化。
3. 页面要有明确主题，不只是空白 demo。
4. 代码适合新手阅读。
5. 最后给我一段简历项目描述。
```

## 让模型解释代码

```text
请用新手能听懂的方式解释这段代码。

要求：
1. 先说整体作用。
2. 再按模块解释。
3. 专业词要翻译成人话。
4. 最后告诉我如果想改效果，应该改哪些地方。
```

## 让模型生成实习项目描述

```text
请把这个网页项目整理成适合简历的项目经历。

输出：
项目名称：
项目简介：
技术栈：
我负责的内容：
项目亮点：
面试时可以怎么讲：

要求表达真实，不要夸大。
```
