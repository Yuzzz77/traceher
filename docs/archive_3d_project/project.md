# 项目背景

## 用户身份

用户是一名在读大二学生，就读于武汉纺织大学信息管理与信息系统相关专业，目前希望寻找实习。

用户目前的主要目标：

1. 提升自己的技术能力。
2. 做出能放进简历和作品集的网站。
3. 用粒子交互、3D、数据可视化等效果增加项目亮点。
4. 学会更高效地使用 ChatGPT、Gemini、Codex、Claude Code 等大模型工具。

## 当前项目方向

这个项目主要用于练习和展示：

- HTML / CSS / JavaScript 网页开发
- 粒子交互系统
- 视觉化交互页面
- 适合实习简历展示的小作品

优先做成这些类型：

- 可直接打开的单页网站
- 有交互效果的网页作品
- 粒子背景或粒子编辑器
- 个人作品集页面
- 数据可视化展示页

## 作品集标准

每个作品尽量做到：

1. 打开就能看到完整效果。
2. 页面有明确主题，不只是技术 demo。
3. 交互有反馈，例如鼠标移动、点击、拖拽、参数调节。
4. 代码结构清楚，新手能看懂。
5. README 或说明里能讲清楚项目亮点。

## 技术选择原则

优先级如下：

1. 原生 HTML / CSS / JavaScript
2. Three.js / Canvas / SVG
3. Vite + React
4. 其他框架或库

不要为了显得高级而引入复杂技术。能用简单方法做出好效果，优先简单方法。
